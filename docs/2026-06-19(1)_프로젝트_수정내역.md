# 2026-06-19(1) 프로젝트 수정내역

## 목적

현재 `heejin` 브랜치의 최신 UI/UX 수정 사항과 전달 문서를 정리한 기록입니다.
이번 작업은 Care Vision 웹앱의 Home, Device, Chat, Self Care, Settings 계열 화면의 시각 일관성과 사용 흐름을 정리하는 데 초점을 두었습니다.

## Chat 화면

- Figma Make 시안 톤을 참고해 채팅 전환 후 화면 디자인을 정리했습니다.
- 기존 UI/UX 흐름과 LLM/API 연동 로직은 유지했습니다.
- Manual Guide 클릭 전에는 가이드 상세가 자동으로 노출되지 않도록 수정했습니다.
- Manual Guide 클릭 시 `/self-care` 이동 대신 `/chat` 안에서 LLM 응답 기반 가이드를 인라인으로 펼치도록 변경했습니다.
- Manual Guide 영상과 텍스트 매뉴얼을 같은 대화 카드 안에 묶어 출력하도록 변경했습니다.
- Manual Guide 카드 배경을 텍스트 매뉴얼 카드 톤과 통일했습니다.
- `Troubleshoot Product Issue` 선택 칩의 체크 아이콘을 복구했습니다.
- 완료 메시지의 줄바꿈 문자를 제거했습니다.
- Chat 첫 진입 모션과 재진입 모션을 분리해, 초기 진입 화면과 진행 중 재진입 화면의 전환을 다르게 처리했습니다.
- Chat 헤더 하단에 흰색 띠처럼 보이던 border를 제거했습니다.

## Home / Device / Product Detail 계열

- Device 화면의 `My Devices` 제목과 첫 카드 사이 간격을 Home 화면의 인사말 아래 간격 기준에 맞췄습니다.
- Settings 화면의 `Settings` 제목 간격도 같은 기준으로 조정했습니다.
- Product Details, Device, Settings 계열 카드 스타일과 제목 스타일을 Home의 Care Risk Score 계열 톤에 맞춰 정리했습니다.
- Care Summary, Product Status, Care History 영역의 카드 배경과 색상 톤을 화면 전반의 글래스/민트 계열 디자인에 맞춰 조정했습니다.

## Self Care 화면

- 제품 카드의 상단 답답함을 줄이고 제목, 모델명, 이미지 배치를 조정했습니다.
- Manual 탭 영상 영역을 카드 섹션 안에 넣어 `Official Video Guide · Filter Cleaning` 형식으로 정리했습니다.
- Manual / AR 탭 글자 크기를 키워 가독성을 개선했습니다.
- AR 탭의 `AR Guide Get Started` 버튼 위치는 원래 순서인 `Steps 설명 카드` 아래로 복구했습니다.

## AR Guide 화면

- AR Guide 헤더의 기존 `AR` 텍스트를 전달받은 SVG 디자인으로 교체했습니다.
- 새 asset 파일을 추가했습니다.
  - `frontend/src/imports/ar-title.svg`

## Settings 계열 화면

- `Language Settings`와 `Edit Profile` 헤더 간격을 Product Details 화면 기준으로 맞췄습니다.
- `Language Settings` 제목의 글자 크기, 색상, 굵기, line-height를 Self Care / Product Details 계열과 맞췄습니다.
- 언어 목록에서 선택된 언어만 볼드로 보이도록 수정했습니다.
- 한국어 항목 표기를 `한국어 / Korean`으로 변경했습니다.
- `Language Settings`와 `Edit Profile`의 Save 버튼을 민트 그라데이션 버튼으로 통일했습니다.

## 백엔드 참고

- Chat의 Manual Guide 클릭은 새 API를 호출하지 않고, 기존 Bot 메시지의 `guideOptions` 데이터를 사용해 인라인으로 펼칩니다.
- 기존 `guide_options`, `manual_guides`, `youtube_recommendations`, `procedure_type`, AR guide 관련 구조는 유지되어야 합니다.
- mock DB 파일도 현재 프로젝트 상태에 포함되어 변경분으로 잡혀 있습니다.
